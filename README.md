# Modern Fitness Landing Page

A modern, responsive landing page built for service-based businesses and personal brands. Designed with a focus on clean UI, clear messaging, and conversion-driven layout.

## Overview

This project showcases a production-ready marketing website built using a modern frontend stack. The layout emphasizes visual hierarchy, strong call-to-actions, and mobile-first responsiveness.

The goal was to create a fast, minimal, and scalable landing page structure that can easily be adapted for different industries (coaching, SaaS, consulting, startups).

## Features

- Fully responsive layout (mobile-first)
- Clean, modern UI with consistent spacing system
- Call-to-action focused hero section
- Services and testimonials sections
- Interactive contact form with success state
- Smooth scroll navigation
- Optimized build using Vite

## Tech Stack

- React (Vite)
- Tailwind CSS
- JavaScript (ES6+)
- Git for version control
- Deployed via Vercel
